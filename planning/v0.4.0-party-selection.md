# v0.4.0 - Party Selection & Variety

**Status:** Completed
**Date:** 2025-12-03

## Objectives
1.  **Party Selection:** Allow players to choose their starting party.
2.  **Variety:** Introduce unique traits and starting conditions for each party.
3.  **Realism:** Use realistic Belgian party archetypes.

## Features Implemented
-   **New Game Phase:** `setup` phase added to `GamePhase`.
-   **Party Selection UI:** `PartySelectionView` component created.
-   **Dynamic Scenarios:** `createScenarioState` factory function implemented.
-   **Party Archetypes:**
    1.  **Socialist Party** (The Pillar)
    2.  **Liberal Reformists** (Blue Factory)
    3.  **Flemish Alliance** (Community First)
    4.  **The Ecologists** (Climate Vanguard)
    5.  **Flemish Interest** (Secure Borders)
    6.  **Workers' Party** (Class Struggle)
